# 🎮 UNO AI

Handleiding voor het starten en gebruiken van de **UNO AI**-simulatie.

---

## 📋 Vereisten

Zorg ervoor dat het volgende is geïnstalleerd:

- Python
- Visual Studio Code (VS Code)

---

## 🚀 De simulatie starten

### 1. Start de computer
Start de computer of laptop op (indien dit nog niet is gedaan).

### 2. Open Visual Studio Code
Start **Visual Studio Code (VS Code)**.

### 3. Open het project
Navigeer in de **Explorer** naar de map:

```text
UNO-AI-MAP
```

Open deze map als project.

### 4. Open het startbestand
Open het bestand:

```text
AI_gameRun.py
```

### 5. Start het programma
Klik rechtsboven in VS Code op de **Run**-knop.

### 6. Gebruik de terminal
Klik onderaan in VS Code op de **Terminal**, zodat je invoer kunt geven.

### 7. Kies het aantal spelers
Voer het gewenste aantal spelers in.

**Toegestaan:**

- Minimaal: **2**
- Maximaal: **10**

### 8. Bekijk de simulatie
De AI speelt nu automatisch het UNO-spel.

### 9. Bekijk de winnaar
Wanneer de simulatie is afgelopen, wordt de winnende speler weergegeven in de terminal.

---

## ⚙️ Optioneel

Wil je de snelheid van de simulatie aanpassen?

Open:

```text
AI_code.py
```

Pas vervolgens **regel 52** aan om de wachttijd tussen de beurten van de spelers te wijzigen.
